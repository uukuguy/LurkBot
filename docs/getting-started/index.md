# Getting Started

Welcome to LurkBot! This section will help you get up and running quickly.

## What is LurkBot?

LurkBot is a Python reimplementation of [moltbot](https://github.com/moltbot/moltbot) â€” a powerful personal AI assistant platform that connects multiple messaging channels to AI models like Claude, GPT, and Gemini.

## Key Features

- **Multi-Channel Support**: Connect WhatsApp, Telegram, Discord, Slack, Signal, and more
- **Multi-Model AI**: Use Claude, GPT, Gemini, or local models via Ollama
- **Tool Execution**: Run shell commands, automate browsers, manage files
- **Sandbox Isolation**: Secure execution in Docker containers
- **Session Management**: Persistent conversations with context tracking
- **Skills System**: Extend functionality with custom plugins

## System Requirements

- **Python**: 3.12 or higher
- **Package Manager**: [uv](https://docs.astral.sh/uv/) (recommended) or pip
- **Docker**: Optional, required for sandbox isolation
- **OS**: Linux, macOS, or Windows (WSL2)
