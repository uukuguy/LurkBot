## Quick Links

### Common Tasks

- **Start chatting**: `lurkbot agent chat`
- **Start the gateway**: `lurkbot gateway start`
- **Check configuration**: `lurkbot config show`
- **Add a Telegram bot**: [Telegram Setup](channels/telegram.md)
