# Design Documentation

This directory contains architecture and design documents for LurkBot.

## Document Organization

All design documents have bilingual versions (English and Chinese):
- Default version (no suffix): **English**
- Chinese version: `.zh.md` suffix

## Available Documents

### ARCHITECTURE_DESIGN.md

**English**: [ARCHITECTURE_DESIGN.md](ARCHITECTURE_DESIGN.md)
**Chinese**: [ARCHITECTURE_DESIGN.zh.md](ARCHITECTURE_DESIGN.zh.md)

### MOLTBOT_ANALYSIS.md

**English**: [MOLTBOT_ANALYSIS.md](MOLTBOT_ANALYSIS.md)

## Contributing

When adding new design documents:
1. Create English version as default (e.g., `NEW_DOCUMENT.md`)
2. Create Chinese version with `.zh` suffix (e.g., `NEW_DOCUMENT.zh.md`)
3. Update this README with links to both versions
